## CHANGE 5 — Replace scrollObserver with scroll event

In the boot section at the bottom, add a scroll event listener 
for loading the next batch of tiles:

  window.addEventListener('scroll', () => {
    if (isLoading || currentPage >= totalPages) return;
    const distanceFromBottom = document.documentElement.scrollHeight 
                             - window.scrollY 
                             - window.innerHeight;
    if (distanceFromBottom < 300) loadNextPage();
  });
